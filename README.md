# React Learning Lab

🎯 Propósito

Este repositório é o meu laboratório pessoal de estudos. Aqui, documento conceitos, projetos e práticas da utilização do **React Native**. 
O objetivo é centralizar conceitos fundamentais, anotações técnicas e pequenos projetos experimentais utilizados no meu aprendizado.

📚 O que você encontrará aqui?

Este laboratório está dividido em três parte:
1. Documentação sobre o funcionamento interno do React, incluindo:  
    - Virtual DOM  
    - Ciclo de vida dos componentes  
    - Imutabilidade  
    - Fluxo de dados unidirecional  
    - Componentização  
    - Conceitos essenciais de JavaScript (ES6+) aplicados ao React  
2. Hooks & State: Exercícios focados em manipulação de estado com `useState`, `useEffect` e criação de Custom Hooks.  
3. Laboratórios Práticos: Projetos iniciais (Miniapps) que aplicam os conceitos de `props`, renderização condicional e consumo de APIs.  

📂 Estrutura do Repositório

```bash
├── 📝 documentacao/      # Anotações sobre a parte conceitual (Markdown)
├── 🧪 experimentos/      # Testes rápidos e componentes
```
